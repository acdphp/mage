# Mage
- Simple game made in pure HTML5 + native javascript.

## Demo
https://acdphp.github.io/mage/

## Goal
- Don't die!
- Get mana from dead zombies.
- Beat your own high score. :)

## Controls
| Key     | Action     |
|---------|------------|
| W       | Move up    |
| S       | Move down  |
| A       | Move left  |
| D       | Move right |
| Cursor  | Aim        |
| Left MB | Fire       |
